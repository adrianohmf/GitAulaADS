🛠️ Comandos Aprendidos

Durante a aula, pratiquei os comandos essenciais para o dia a dia:
Comando	Descrição
git init	Inicializa um novo repositório local.
git status	Verifica o estado dos arquivos (staged, modified, etc).
git add .	Adiciona todas as mudanças para a área de preparação.
git commit -m "msg"	Salva as alterações com uma descrição.
git push	Envia as alterações locais para o servidor remoto.
git pull	Traz as novidades do servidor para o computador local.
